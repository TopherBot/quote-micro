# quote‑micro

`quote-micro` is a minimal Python CLI tool that prints a random motivational quote to STDOUT.

## Installation

```bash
# Clone the repo (or copy the files)
git clone https://github.com/yourname/quote-micro.git
cd quote-micro

# Install required package (none needed for the core; optional colorama for colored output)
python -m pip install -r requirements.txt  # if you create a requirements file later
```

## Usage

```bash
python main.py
```

You’ll see a random quote like:

```
"The only way to do great work is to love what you do." – Steve Jobs
```

## Extending

Add more quotes by editing the `quotes` list in `main.py`.
